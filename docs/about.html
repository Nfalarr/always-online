About Always Online LITE

The story behind the project

Project Purpose

Always Online LITE is an educational tool designed to demonstrate how Discord's Gateway API works. This web version was created to help developers understand real-time WebSocket communication with Discord's services.

This project shows how presence updates are managed through Discord's Gateway using direct WebSocket connections, maintaining online status and custom activities.

Technology

This tool uses modern web technologies to establish a direct connection to Discord's Gateway:

WebSocket Protocol
Vanilla JavaScript
JSON payload handling
Heartbeat mechanisms
Auto-reconnection
Security measures

How It Works

Here's a simplified example of how the connection to Discord's Gateway works:


// Initialize WebSocket connection to Discord Gateway
const gateway = new WebSocket('wss://gateway.discord.gg/?v=9');

// Event handler for successful connection
gateway.onopen = () => {
  console.log('Connected to Discord Gateway');
  
  // Send IDENTIFY payload with authentication token
  gateway.send(JSON.stringify({
    op: 2,
    d: {
      token: 'YOUR_TOKEN',
      properties: {
        os: 'windows',
        browser: 'chrome',
        device: ''
      },
      presence: {
        status: 'online',
        activities: [{
          name: 'Your Activity',
          type: 0
        }]
      }
    }
  }));
};

// Implement heartbeat mechanism to keep connection alive
setInterval(() => {
  gateway.send(JSON.stringify({
    op: 1,
    d: null
  }));
}, heartbeat_interval);
